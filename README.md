# Dream Something

###### Matt Czencz

### Tech used:

- Vite
- React
- Sass / SCSS
- Framer Motion
- Node version `20.13.0` (18 and greater should work)

### Installation steps:

1. Clone this repo
2. Run `yarn` to install dependencies
3. Run `yarn dev` to start up local environment
